# Lead Management Dashboard (Healthcare Analytics)

## Overview
This project presents a **Lead Management Dashboard** designed for a healthcare use case, built to analyze patient flow, billing performance, diagnosis trends, and operational efficiency.  
The dashboard converts raw hospital data into **actionable business insights** using interactive visuals and KPIs.

## Objective
The primary goal of this dashboard is to:
- Monitor patient admissions and follow-ups
- Track billing and insurance amounts
- Analyze diagnosis-wise patient distribution
- Evaluate bed occupancy across hospital units
- Assess doctor-wise feedback performance

## Key Metrics (KPIs)
- **Admit Date**
- **Discharge Date**
- **Follow-Up Date**
- **Total Billing Amount (190M)**

## Dashboard Insights
### Patient & Operations
- **Bed Occupancy Analysis** across:
  - Private
  - General
  - ICU
- Helps identify utilization patterns and capacity planning opportunities

### Diagnosis Analysis
- Diagnosis-wise patient count including:
  - Viral Infection
  - Flu
  - Malaria
  - Typhoid
  - Pneumonia
  - Fracture
- Funnel-style visualization highlights case drop-offs and prevalence

### Financial Analysis
- Comparison of:
  - **Total Billing Amount**
  - **Health Insurance Amount**
- Diagnosis-wise revenue trends enable financial forecasting

### Doctor Performance
- Feedback value distribution per doctor
- Helps management identify consistency and performance balance
